# 🌌 Атлас нерешённых проблем физики и космологии

Набор из **23 самостоятельных HTML-страниц** — визуальных объяснений главных открытых
загадок современной физики: от того, куда гравитация тянет нашу Галактику, до вопроса,
из чего вообще соткано пространство-время. Собран в ходе одного разговора-путешествия
«от космоса вокруг нас — вглубь, к основаниям реальности».

## Как пользоваться

1. Открой **`atlas-hub.html`** в любом браузере (двойной клик).
2. Кликай по карточкам — они ведут на соседние файлы в этой же папке.

Всё работает **офлайн**, без интернета: каждый файл самодостаточен (стили встроены,
внешних зависимостей нет). Просто держи все файлы в одной папке.

## Технические детали

- Кодировка **UTF-8**, `<!doctype html>`, адаптивная вёрстка (читается и с телефона).
- Тёмная тема выбрана намеренно — под цвет предмета.
- Уважается `prefers-reduced-motion` (для чувствительных к анимации).

## Оговорка

Материал научно-популярный: цифры округлены, а схемы, кривые и диаграммы —
**иллюстративные**, а не точные графики. Статусы гипотез и значения (σ, проценты)
актуальны на **июль 2026 года**; передний край физики меняется быстро.
